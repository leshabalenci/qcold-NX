# qColdNX by [lesha aka sexyxxxman](https://leshayyilt.t.me)
![](https://raw.githubusercontent.com/leshabalenci/qColdNX/main/qco1ld.png)

### qCold - лучшая для игры прошивка для нинтендо свитч с разгоном и гибкой настройкой

## Состав qCold
1. **[Atmosphere](https://github.com/Atmosphere-NX/Atmosphere)** - спизженая база под прошивку
1. **[Hekate](https://github.com/CTCaer/hekate)** - бутлоудер


1. **Установленные пейлоады**:
   * [Lockpick RCM](https://github.com/impeeza/Lockpick_RCMDecScots) - чтоб пиздить прод кейс с консоли
   * [TegraExplorer](https://github.com/suchmememanyskill/TegraExplorer) - какая-то хуйня ее все ставят не ебу зачем она тут)

1. **Установленные Homebrew**
   * [Sphaira](https://github.com/ITotalJustice/sphaira) - лучше чем хбменю (не слушайте америкозов которые пиздят что хбменю лучше чем сфайра)
   * [AiO Switch Updater](https://github.com/HamletDuFromage/aio-switch-updater) - программа для обновы qColdNX и хуемое ебал рот 
   * [Daybreak](https://github.com/Atmosphere-NX/Atmosphere) - обновление ПО свитча
   * [DBI](https://4pda.to/forum/index.php?showtopic=939714&st=1100#entry86288632) - заебись темка там игры качаешь хуемое
   * [Fizeau](https://github.com/averne/Fizeau) - меняй цвет теплоту хуемое экрана заебись мне нраица
   * NxThemesInstaller - инжект тем на свитче
   * [ThemezerNX](https://github.com/ThemezerNX/Themezer) - браузер тем разных
   * [Linkalho](https://github.com/HamletDuFromage/linkalho) - фейк привязка к NNID чтоб заходить в эмуляторы и тд

1. **Установленные модули**
   * [SaltyNX](https://github.com/masagrator/SaltyNX) - модуль который разрешает редактировать хуемое не разобрался
   * [sys-clk](https://github.com/ppkantorski/sys-clk) - разгон процессора видеочипа оперативы заебись крч
   * [nx-ovlloader](https://github.com/ppkantorski/nx-ovlloader) - заебца модуль хз зачем он
   * qColdHand Overlay - заебись оверлей мне нравится
     - [sys-clk](https://github.com/ppkantorski/sys-clk) - оверлей разгона 
     - [Status Monitor](https://github.com/ppkantorski/Status-Monitor-Overlay), форк [Status Monitor](https://github.com/hanai3Bi/Status-Monitor-Overlay) - статус монитор хуй знает как назвать
     - [FPSLocker](https://github.com/masagrator/FPSLocker) - анлок фпс в играх до 60
     - [ReverseNX-RT](https://github.com/masagrator/ReverseNX-RT) - смена настройки графы и разрешения путем фейкового дока/портатива
     - [Sysmodules](https://github.com/ppkantorski/ovl-sysmodules) - включение/выключение системных модулей
